# Concierge

Signalling server backend for Tandem


## Quickstart

The signalling server, written in Java 8 and Spring.
It uses WebSockets to facilitate clients and hosts in exchanging offers and answers.

## Development

1. `cd concierge`
1. `gw bootRun`

## Deployment

1. `cd concierge`
1. `gw build`
1. `cf push`
